Trait representing an ELF File Header.

This trait allows HeaderRepr to provide a consistent API regardless of whether the header section uses 32- or 64-bit ELF words. A number of field values in the header of various sizes are converted to usize by this API so that they can be used as indices, etc.

For more information on ELF File Headers, refer to: + the ELF [specification] + the [OS Dev Wiki]
