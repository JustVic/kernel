Module cpu::segment [−] [src]

Code for using the x86 and x86_64 segmentation hardware.

For more information, refer to the Intel® 64 and IA-32 Architectures Software Developer’s Manual, Vol. 3A, section 3.2, "Using Segments". Some of the documentation present in this module was taken from the Intel manual.
